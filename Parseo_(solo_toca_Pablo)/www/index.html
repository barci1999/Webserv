Bienvenido a tu Webserv

Esta es la página principal de tu proyecto webserv de 42 School.

Desde aquí puedes navegar a las páginas de prueba de los diferentes métodos HTTP, ver errores o incluso divertirte con el parrot.

Páginas de prueba

  • GET — Prueba de recuperación de recursos.
  • POST — Formulario tipo foro.
  • DELETE — Prueba de eliminación de recursos.
  • Parrot — Comando curl divertido.

Páginas de error

  • 400 - Bad Request
  • 404 - Not Found
  • 405 - Method Not Allowed

Estas páginas se mostrarán automáticamente cuando el servidor detecte errores.
